# DeepRacer Scripts

Scripts to setup and manage sets of SageMaker Notebook instances. 

The following scripts are included:

- `sagemaker-notebook-create-multiple.py`: Create several SageMaker Notebook instances at once
- `sagemaker-notebook-delete-one.py`: Delete a single SageMaker Notebook instance
- `sagemaker-notebook-delete-all.py`: Delete all SageMaker Notebook instances in a region
- `sagemaker-notebook-link-one.py`: Create a signed URL for one SageMaker Notebook instance
- `sagemaker-notebook-links-all.py`: Create signed URLs for all SageMaker Notebook instances in a region
- `sagemaker-notebook-start-all.py`: Start all SageMaker Notebook instances in a region
- `sagemaker-notebook-start-one.py`: Start a single SageMaker Notebook instance
- `sagemaker-notebook-stop-all.py`: Stop all SageMaker Notebook instances in a region
- `sagemaker-notebook-stop-one.py`: Stop one SageMaker Notebook instance
